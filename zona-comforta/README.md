## Build commands
- Android
```bash
flutter build appbundle -t lib/main_seller.dart --flavor seller 
flutter build ipa -t lib/main_buyer.dart --flavor buyer
```
- IOS
```bash
flutter build ipa -t lib/main_buyer.dart --flavor buyer
flutter build ipa -t lib/main_seller.dart --flavor seller
```
